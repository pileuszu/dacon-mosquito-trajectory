## 🎯 대회 개요
### 1. 배경 및 목적
*   LiDAR 시스템에서 발생하는 **80ms의 고정 지연 시간** 동안 이동하는 모기의 위치를 예측.
*   탐지 시점의 좌표가 아닌, 실제 조사 시점(+80ms)의 좌표를 예측하여 명중률을 높이는 것이 핵심.

### 2. 주요 과제
*   과거 400ms 동안의 3차원 좌표 시계열 데이터를 바탕으로 미래 위치(+80ms) 예측.
*   실내, 복도, 창고, 야외 등 다양한 환경의 비행 패턴과 노이즈 대응.

---

## 📊 데이터 명세
### 1. 입력 및 출력
*   **입력 데이터**: 과거 11개 시점의 3차원 좌표 (40ms 간격)
    *   `-400ms, -360ms, ..., -40ms, 0ms` (총 11개 포인트)
*   **출력 데이터**: 마지막 관측 시점(0ms) 기준 **+80ms 시점**의 3차원 좌표 `(x, y, z)`

### 2. 좌표계 정의 (Sensor-local)
*   **단위**: 미터 (m)
*   **축 방향**: `x`: forward, `y`: left, `z`: up

## 📏 평가 산식 (Metric)
### **R-Hit@1cm**
예측 좌표와 실제 좌표의 3차원 유클리드 거리 $d$를 계산하여, 레이저 유효 범위($d \le 0.01m$) 내에 들어오면 적중으로 판정합니다.

*   **판정 기준**: $d \le 0.01m \rightarrow 1$ (적중), $d > 0.01m \rightarrow 0$ (실패)
*   **최종 점수**: 전체 샘플의 평균 R-Hit(적중률)

```python
import numpy as np

def r_hit(pred, true):
    R_HIT = 0.01 # 1cm
    distance = np.linalg.norm(np.asarray(pred) - np.asarray(true), axis=-1)
    return np.mean(distance <= R_HIT)
```
