# 모기 궤적 예측: 비행 패턴 군집화(Clustering) 및 베이지안 혼합 모델(BGM) 심층 분석 보고서

본 보고서는 전체 학습 데이터(10,000개 궤적)의 비행 역학(Flight Kinematics) 지표를 분석하여 비행 패턴 군집화를 적용하고, 전문가 혼합 모델(Mixture of Experts)의 설계 방향을 검증한 종합 보고서입니다.

## Part 1. 가우스 혼합 모델(GMM) 기반 4개 비행 모드(Regime) 분류

### 1.1 4개 군집 분할(K=4)의 수학적 최적성 검증
학습 데이터의 비행 속도, 가속도, 곡률, 횡가속도, Z축 연직 속도/가속도, Saccade 확률 등 8차원 물리 피처 공간에서 군집 수($K$)에 따른 관성(Inertia) 및 실루엣 계수(Silhouette Score)를 측정했습니다.

| 군집 수 ($K$) | 관성 (Inertia, 낮을수록 우수) | 실루엣 계수 (Silhouette Score, 높을수록 우수) | 평가 및 의미 |
| :---: | :---: | :---: | :--- |
| **K = 2** | 62,441.67 | 0.2752 | 느린 비행 vs 빠른 비행의 거시적 분할 |
| **K = 3** | 54,372.64 | 0.2480 | 과도기적 수렴 상태 (불안정) |
| **K = 4** | **48,570.25** | **0.2558** | **로컬 피크(Local Peak) 발생 - 4개 패턴 분류 타당성 입증** |
| **K = 5** | 43,519.50 | 0.2498 | 군집 과분할 및 해석력 저하 |

*   **해석**: $K=2$에서 속도 기준의 큰 분할이 일어난 후, **$K=4$에서 실루엣 계수가 다시 상승하는 로컬 피크(0.2558)**가 관측되었습니다. 이는 10,000개 모기 궤적이 수학적·물리적으로 4개의 뚜렷한 거동 패턴 구조를 스스로 형성하고 있음을 뜻합니다.

### 1.2 경계선 분류 모호성 검증 (Boundary Ambiguity)
가우시안 혼합 모델(GMM)을 활용하여 각 궤적이 4개 군집에 속할 사후 확률(Posterior Probability)을 계산하고, 경계선 데이터의 비율을 추출했습니다.
*   **명확한 분류 (Clear Trajectories)**: 특정 군집에 속할 확률이 **85% 이상**인 궤적
*   **애매한 경계선 분류 (Ambiguous Trajectories)**: 가장 확률이 높은 군집의 확률이 **85% 미만**인 궤적
*   **검증 결과**:
    *   명확하게 분류되는 궤적: **8,997개 (89.97%)**
    *   경계선에 걸쳐진 모호한 궤적: **1,003개 (10.03%)**
*   **결론**: 약 90%의 데이터가 물리적으로 매우 뚜렷한 거동 의도(직진/선회/속도)를 가지고 비행하고 있음이 입증되었습니다. 10% 내외의 모호한 전이 상태(Transition State) 데이터가 존재하지만, 대다수의 궤적은 4개 카테고리로 깨끗하게 분리되므로 4개 개별 전용 모델로 분할 학습시키는 것이 타당합니다.

## Part 2. 베이지안 가우시안 혼합 모델(BGM) 자동 비행 패턴 군집화 및 12-Expert 설계

### 2.1 핵심 요약 (Executive Summary)
*   **최적 군집 개수 자동 산출**: 디리클레 프로세스 사전 확률(Dirichlet Process Prior) 기반 학습 결과, 총 12개의 최대 컴포넌트 상한선 중 **12개 모두 유효 가중치(Weight > 1.0%)를 가진 활성 군집(Active Clusters)**으로 스스로 선택되었습니다.
*   **미세 비행 상태 포착**: 기존 GMM-4 구조에서는 속도나 선회력 차이가 큰 궤적들이 뭉뚱그려졌으나, BGM은 **속도 0.74 cm/s, 곡률 109.25**의 극도로 특이한 제자리 호버링 선회 비행(Cluster 5, 2.03%) 등 생물학적으로 고유한 12대 미세 패턴을 자율적으로 분리해 냈습니다.
*   **경계선 데이터 탐지 (95% 신뢰도 기준)**:
    *   특정 군집에 확실히 소속되는 궤적 (Clear, >= 95%): **6,694개 (66.94%)**
    *   경계선에 걸쳐진 애매한 궤적 (Ambiguous, < 95%): **3,306개 (33.06%)**
    *   군집 수가 12개로 미세화되면서 다차원 경계 영역의 밀도가 늘어나 경계선 분류 비율(33.06%)이 상승했습니다. 이는 학습 시 의사결정 나무 경계면의 왜곡을 방지하기 위해 **이상치 필터링(Denoising)**을 엄격하게 적용해야 함을 입증합니다.

### 2.3 12-Expert Denoised Training 구현 설계
*   **학습 데이터 노이즈 제거 (Denoising)**: 12-BGM의 각 클러스터에 대한 사후 확률 중 최대 확률 $\max(P_i) \ge 95\%$인 명확한(Clear) 트랙만 필터링하여 학습 데이터로 활용합니다. 경계선에 애매하게 걸친 데이터(33.06%)를 제거하여 의사결정 나무(Decision Tree)의 분할 경계면이 물리적으로 겹치지 않고 정밀하게 학습되도록 유도합니다.
*   **12개 전문가 모델 분할 훈련**: 각 12개 클러스터 전용 tabular 예측기를 독립적으로 학습시킵니다. 단, 샘플 수가 극단적으로 적은 군집은 과적합을 방지하기 위해 Random Forest 및 Extra Trees 모델을 제외하여 메모리 세이프티를 확보합니다.
*   **추론 및 Soft Blending**: 테스트 케이스 후보군에 대해 12개 전문가 모델의 연속 거리 예측값 $\hat{d}_{0} \sim \hat{d}_{11}$을 구한 뒤 BGM 확률 $P_0 \sim P_{11}$을 가중치로 삼아 소프트 블렌딩을 수행합니다:
    $$P_{\text{final}}(\text{candidate}) = \sum_{k=0}^{11} P_k \times \exp\left(-\frac{\hat{d}_k}{\sigma}\right)$$
    이를 통해 경계선 영역(33.06%)의 궤적도 12개 전문가들의 기여도로 부드럽고 정확하게 복원됩니다.
